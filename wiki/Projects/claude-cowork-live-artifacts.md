# Claude Cowork / Live Artifacts

## Objetivo

Guía/clase tutorial sobre cómo crear dashboards de negocios en vivo usando Claude + Google Sheets (MCP), sin código. Orientada a emprendedores no técnicos.

## Material creado

- Presentación Google Slides: "Claude Live Artifacts: Guía de Clase Tutorial" (13 slides)
- Basada en tutorial de Ryan & Matt Data Science

## Concepto central

**Artifacts estáticos vs Live Artifacts:**
- Estáticos = foto congelada, se vuelven obsoletos
- Live Artifacts = app real conectada a Google Sheets, se actualiza automáticamente

## Flujo de 4 pasos enseñado

1. Conectar Google Sheets via MCP
2. Escribir prompt descriptivo
3. Claude renderiza la web app
4. Iterar con filtros, fechas, etc.
