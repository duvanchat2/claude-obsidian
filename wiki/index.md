# Brain — Índice Principal

Este vault sigue la metodología **PARA** (Projects / Areas / Resources / Archives) para organizar todo el conocimiento y proyectos.

## Estructura

| Carpeta | Qué va aquí |
|---------|-------------|
| Projects | Proyectos activos con un objetivo y fecha de entrega |
| Areas | Responsabilidades continuas sin fecha de fin (ej: marketing, finanzas) |
| Resources | Referencias, guías, aprendizajes reutilizables |
| Archives | Proyectos terminados, pausados o descartados |

## Cómo usar

1. **Nuevo proyecto** → crea una nota en `Projects/` usando la plantilla de proyecto
2. **Ingesta de fuente** → suelta el archivo en `.raw/` y di "ingest [nombre]"
3. **Pregunta** → "query: [tu pregunta]" y Claude lee el wiki primero
4. **Lint** → "lint the wiki" cada 10-15 ingestas
